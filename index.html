CedToCal

Cédula a Cumpleaños en Google Calendar.

Sube tu archivo Excel/CSV, extrae las fechas de nacimiento a partir de las cédulas nicaragüenses y agenda tus recordatorios automáticamente en Google Calendar.

Privacidad: tu archivo se procesa en tu propio dispositivo. Lo único que sale a internet son los cumpleaños que tú decides agregar a tu calendario.
⚠️ Falta configurar el ID de cliente

No se encontró el GOOGLE_CLIENT_ID. Copia config.example.js a config.js, pega tu ID y sirve la página por http:// (p. ej. python -m http.server 8000).

Cómo obtener el ID de cliente (una sola vez, gratis)
  1. Ve a console.cloud.google.com → crea un proyecto.
  2. APIs y servicios → Biblioteca → habilita Google Calendar API.
  3. Pantalla de consentimiento de OAuth → Externo → agrégate como usuario de prueba.
  4. Credenciales → Crear credenciales → ID de cliente de OAuth → Aplicación web.
  5. En Orígenes autorizados de JavaScript agrega http://localhost:8000.
  6. Pega el ID en config.js (copiado de config.example.js).
