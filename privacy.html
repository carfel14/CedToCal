Política de Privacidad

CedToCal — Cédula a Cumpleaños en Google Calendar

Última actualización: 24 de junio de 2026

CedToCal ("la aplicación") es una herramienta web que lee un archivo de Excel o CSV con nombres y cédulas nicaragüenses, deriva la fecha de nacimiento contenida en cada cédula y la agenda como evento anual en tu Google Calendar. Esta política explica qué datos se usan y cómo.

1. Procesamiento local del archivo

El archivo que subes (nombres y cédulas) se procesa íntegramente en tu propio navegador. Su contenido no se envía a ningún servidor nuestro: no operamos servidores que reciban, almacenen o procesen tus archivos. La aplicación es un sitio estático sin backend.

2. Datos de Google que utiliza la aplicación

Con tu autorización explícita (inicio de sesión con Google), la aplicación accede a la API de Google Calendar mediante el permiso https://www.googleapis.com/auth/calendar con un único fin:

  • Consultar la lista de tus calendarios para encontrar o crear el calendario de destino (por defecto "Cumpleaños").
  • Crear eventos de cumpleaños en ese calendario.

La aplicación no lee, modifica ni elimina otros eventos ni otros datos de tu cuenta, y solo actúa cuando tú pulsas "Agregar al calendario".

3. Almacenamiento y conservación

No almacenamos tus datos. El token de acceso de Google se mantiene únicamente en memoria durante tu sesión en el navegador y se descarta al cerrar la pestaña. No usamos cookies de seguimiento ni bases de datos.

4. Compartición con terceros

No vendemos, cedemos ni compartimos tus datos con terceros. La única comunicación de red es directamente entre tu navegador y las API de Google, necesaria para crear los eventos que tú solicitas.

5. Uso limitado (Google API Services User Data Policy)

El uso que CedToCal hace de la información recibida de las API de Google se ajusta a la Política de Datos de Usuario de los Servicios de API de Google, incluidos sus requisitos de Uso Limitado. No usamos los datos de Google para publicidad ni para ningún fin distinto de proporcionar la función descrita en esta política.

6. Revocar el acceso

Puedes retirar el permiso concedido a la aplicación en cualquier momento desde la página de permisos de tu cuenta de Google.
